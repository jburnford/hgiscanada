Canadian Census Subdivisions, 1851–1921

This site publishes one page per Ontario Census Subdivision per census year, drawn from the Canadian Peoples / TCP boundary files and tabulations. Each page combines:

  • A natural-language summary of the place in that year
  • Population and sex split, where the census recorded them
  • A cross-year population trajectory — the same place across all eight censuses, stitched together by spatial polygon overlap
  • Neighbouring CSDs in that census year
  • Wikidata grounding (where available) and persistent place identifiers
  • Source citation back to the original census tables on Borealis
  • Schema.org structured data for search engines and citation-grounded RAG

Why this exists

Historical census data for Canada is maintained as zip files on the Borealis Dataverse. Default-mode chatbots like Gemini and Claude can't read those, can't query a SPARQL endpoint, and can't open a graph database. They can read indexed web pages with prose plus structured data — so this site is the bridge between research-quality structured data and citizen-facing question answering.

Data, code, methodology

This site is generated from the Canada History Knowledge Graph project. The underlying knowledge graph is published as:

  • A CIDOC-CRM Turtle file (for the LINCS LOD ecosystem)
  • A KuzuDB graph database (for researchers using coding agents)
  • These web pages (for citizen chatbot answers)

All three derive from the same source data; this site is the citizen-facing surface.

How identity works here

Each page describes a Place (the enduring entity, e.g. "Westmeath") in a specific Presence (its 1871 census manifestation). Identity across years is established by spatial polygon overlap, not by name match — so a township that was renamed or whose neighbours shifted still threads through to the right modern entity. Where a Place has been linked to Wikidata, this page treats Wikidata's QID as the canonical identifier; queries about the deep-time meaning of a place inherit Wikidata's depth (or its limits).
